# Agent Namespace Conventions

This repository uses specific directories for different agent and AI-first tooling purposes. Please adhere to these conventions:

- `.claude/`: Contains core skills, hooks, and configuration files for Claude/Morty.
- `.jules/`: Contains journal entries, memory tracking, and notes specific to Jules/Bolt performance optimizations.
- `.aifirst/`: Used for broader AI-first tooling, planning, and synthesis artifacts.
